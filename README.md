# dotfiles
My personal dotfiles

## Running the script
To install the environment run:
```shell
make
chsh -s $(which zsh)
```

## Neovim
Running Neovim version `0.10.4`.

## Tmux extensions
Install by opening Tmux and running Ctrl + Space(The prefix key) followed by I(capital i).

## Font of choice
Fonts compatible with the themes: [Nerdfonts](https://www.nerdfonts.com/font-downloads).
